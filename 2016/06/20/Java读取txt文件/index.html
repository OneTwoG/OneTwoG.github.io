title: Java读取txt文件，并对读取字符串进行求和操作
date: 2016-06-15
category:

  • java基础

题目： 从一个txt文件中读取其中的数据。txt文本数据示例如下
8
1 2 3 4 5 6
读取之后分别求出奇数和 偶数和

实例代码

import java.io.FileReader;
class test{
public static void main(String[] args) {

    String getNum = "";


    FileReader readFile = null;

    try{
        readFile = new FileReader("test.txt");

        int ch;

        while ((ch = readFile.read()) != -1) {
//                System.out.print((char)ch);
            getNum += (char)ch;
        }
    }catch(Exception e){
        e.printStackTrace();
    }finally {
        try{
            if(readFile != null) readFile.close();
        }catch(Exception e){
            e.printStackTrace();
        }
    }

    System.out.println(getNum);


    int sumOdd = 0;
    int sumEven = 0;

    for (int i = 2; i < getNum.length(); i+=2) {
//            int temp = getNum.charAt(i) - '0';
        int temp = Integer.parseInt(String.valueOf(getNum.charAt(i)));
        System.out.println(temp);
        if( temp % 2 == 0 ){
            sumEven += temp;
        }else {
            sumOdd += temp;
        }
    }

    System.out.println("奇数和：" + sumOdd);
    System.out.println("偶数和：" + sumEven);
}
}
